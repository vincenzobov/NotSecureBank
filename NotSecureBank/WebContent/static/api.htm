API Catalog

REST APIs are a widespread and growing method to offer operations and data access to front-end application and/or third parties' systems. With our wonderful REST API, you can develop custom applications able to interact with No Secure Bank services.

Base URL: http://$hostname/notsecurebank/api

Login

Login mechanism.

GET /login

Check if any user is loggedin. If a user is loggedin the username will be returned.

Parameters

No parameters.

Responses

CodeDescription
200Status on whether the user is logged in or not.

POST /login

Login method. After a login is successful a success message will be returned.

Parameters

NameDescription
body (required, JSON, request body) { "username":"jsmith", "password":"demo1234" }

Responses

CodeDescription
200Success message when login is complete.
400User error message.
500Server error message.

Account

View account information.

GET /account

Returns a list of all the accounts owned by the user.

Parameters

No parameters.

Responses

CodeDescription
200Successful operation.
401Unauthorized request.
500Internal server error.
